## Backend to administrate users

**CRUD Functionality:**  
  
 - Show all users in DB,  
 - add new user,
 - delete user,  
 - change user,  
 - get user.

**Explore:**   
- Install local MongoDB server and start it,  
- Install Node.js,
- clone git repo to your computer and cd into cloned directory,
- Enter "npm install" in bash/shell/powershell commander,
- Start project or run it in dev mode (npm start / npm run dev).
